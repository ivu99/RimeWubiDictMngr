# Rime五笔码表管理器

## 介绍

本工具用来管理Rime五笔输入法的码表。

Rime的五笔输入法有个优点，它的码表中包含权重（weight）信息。Rime五笔的码表文件`wubixx.dict.yaml`中的条目如下：

```yaml
剖	ukj	113738
问题	ukjg	20455144
总是	ukjg	2866329
总量	ukjg	751184
剖	ukjh	90990
```

每一行记录依次是：词（text），编码（code）和权重（weight），中间用制表符`\t`隔开。当出现重码时，例如，编码`ukjg`对应三个词，“问题”一词的权重最大，会排在最前面。

当前主要有两个功能：

1. 生成基础核心词库
2. 添加用户自定义词库
